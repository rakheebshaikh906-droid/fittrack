# FitTrack 💪

A simple Gym Workout Tracker built using HTML, CSS, and JavaScript.

## Features

- Add workouts
- Track sets, reps, and weight
- Delete workouts
- LocalStorage support
- Responsive dark UI

## Technologies Used

- HTML
- CSS
- JavaScript
